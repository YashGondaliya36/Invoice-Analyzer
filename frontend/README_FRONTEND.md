# Invoice Analyzer Frontend

## Industrial Finance Theme
This frontend is built with a bespoke **"Industrial Finance"** aesthetic, featuring:
-   **Deep Stone Palette**: `#0c0a09` backgrounds for visual comfort.
-   **International Orange**: High-contrast action elements.
-   **Neural Background**: Interactive canvas visualization.
-   **Holographic Scanner**: Advanced drag-and-drop experience.

## Key Features
1.  **Ingestion Bay**: Drag & drop PDF/Images with sci-fi scanning effects.
2.  **Live Ledger**: Real-time grid view of extracted data.
3.  **IO Analyst**: Chat-based AI analysis with Plotly visualizations.

## Development Notes
-   **Schema v2**: As of the latest update, the data model uses `snake_case` keys (`invoice_no`, `total_price`) instead of legacy "Invoice No".
-   **Resync Required**: If you have old sessions from before this update, please **Upload New Invoices** to ensure data compatibility.

## Tech Stack
-   React 19, Vite, Tailwind v4
-   Framer Motion (Animation)
-   Plotly (Visualization)
-   Lucide React (Iconography)
